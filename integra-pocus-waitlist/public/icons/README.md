# Ícones PWA

Coloque aqui os ícones do app para "Adicionar à tela inicial":

- **icon-192.png** — 192×192 px
- **icon-512.png** — 512×512 px

Se não tiver ícones ainda, pode usar um gerador (ex.: [realfavicongenerator.net](https://realfavicongenerator.net/)) a partir de um logo. O app funciona sem eles; o navegador usará um ícone padrão.
